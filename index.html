mathscript
mathscript is a javascript library for converting simple mathscript expressions into MathML HTML elements.
this is originally a school project for maths class, and it is going to be far from perfect for a long time. i will try to keep maintaining this script, but i cannot guarantee updates until the end of time :) thanks for understanding!

how to use

you can use the input box above to enter mathscript expressions, and the result will be shown below. it is that simple! but, here are some basics.
you may use:
  • numbers from 0 to 9
  • lowercase letters from a to z for variables
  • operators + - * (or ×) and / (or ÷)
  • multi-level brackets ( & )
  • fractions in the FRAC{TOP EXPRESSION:FRAC:BOTTOM EXPRESSION}FRAC format: FRAC{x:FRAC:2}FRAC for x2
  • powers in the POW{EXPRESSION:POW:EXPONENT}POW format: POW{x:POW:2}POW for x2
  • roots in the ROOT{BASE:ROOT:NUMBER}ROOT format: ROOT{3:ROOT:125}ROOT for 1253
  • line breaks using \n
  • π using π or PI
